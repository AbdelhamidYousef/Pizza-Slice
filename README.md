# PizzaSlice

This is a demo of an online pizza restaurant website built with React, React Router, Redux Toolkit, Tailwindcss, Vite, and EsLint.

**Technologies used:**

- React
- React Router
- Redux Toolkit
- Tailwindcss
- Vite
- EsLint
